  HELLO WORLD EXAMPLE
  ===================
  ```
  /** 
   * This is a program that prints out a string "Hello World"
   * hello.c
   */
  #include <stdio.h>

  int main() {
    char* my_string = "World";
    printf("Hello %s", my_string);
    return 0;
  }
  ```

  To compile and run this program, 
  if you are on a UNIX system and have gcc installed:
  
  $ gcc -o hello hello.c
  $ ./hello
  Hello World
